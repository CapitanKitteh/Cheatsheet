Informacje ogólne

            Syntax:
            nazwaselektora {
              atrybut: wartość;
            }
            Nazwa selektora to nazwa tagu/elementu, dla id oznaczanego "#" a dla class "." 
            Kaskadowość oznacza odczytywanie reguł od góry do dołu, więc można gdzieś umieścić informację, o szarym kolorze tła dla nagłowka a w dowolnym miejscu niżej określić to kolor czerwony dla tła nagłówka i czerwony będzie wyświetlany.
            nadpisać można wszelkie zasady stosując "!important"
            Umieszczanie osobnego pliku css w pliku html: 
            <link rel="stylesheet" type="text/css" href="nazwapliku.css" />
