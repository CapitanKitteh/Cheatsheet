Why did I make this project:

Project Cheatsheet was made to document my learning progress and help other learn better too.
I'm getting started as complete amateur, without doing anything related to programming in my past, except for knowing the stereotype that programmers are real and they're hacking government institutions :).
Now, more aware, with my first goals all set I'm getting into learning trough courses and other sources
Please, don't treat it like a form of teaching, those are my notes, not a specific descpription about what it contains!
It takes time to learn, but new knowlege and changes are added immediately to this project.

What's inside:

For now, learning plan include Frontend Web Development, but for sure you can find more informations about many different things here in future, so keep watching :)

    Why am I doing this and how can u use it?

  • Memory can fail, did you ever forgot about a little detail or you were not sure how code should look?
  • Just like me, you're new to this and wonder how to document what you will learn?
  • Don't want to waste time looking for code and wondering how would it looked used?
  • After some time you don't want to go back to courses/books to remind basics?

Witaj! Pozwól, że wprowadzę Cię do mojego projektu

Po co ten projekt istnieje:

Projekt Cheatsheet został utworzony, abym mogła udokumentować swoje postępy w nauce i pomagać innym w łatwiejszym przyswajaniu wiedzy.
Zaczynam naukę programowania jako absolutna amatorka, nie mając w przeszłości styczności z czymś więcej poza wiedzą, że programiści istnieją i włamują się do rządowych instytucji :).
Bardziej świadoma, z określonymi pierwszymi celami zabieram się za naukę przerabiając materiały z kursów i innych źródeł.
Proszę, nie traktuj tego jako formy nauczania, to moje notatki a nie szczegółowe przedstawienie materiału.
Nauka zajmuje mi dużo czasu, ale nowa wiedza i zmiany są na bieżąco wprowadzane do projektu.
