Informacje ogólne

            Syntax:
            nazwaselektora {
              atrybut: wartość;
            }
            Nazwa selektora to nazwa tagu/elementu, dla id oznaczanego "#" a dla class "." 
            Kaskadowość oznacza odczytywanie reguł od góry do dołu, więc można gdzieś umieścić informację, o szarym kolorze tła dla nagłowka a w dowolnym miejscu niżej określić to kolor czerwony dla tła nagłówka i czerwony będzie wyświetlany.
            nadpisać można wszelkie zasady stosując "!important"
            Umieszczanie osobnego pliku css w pliku html: 
            <link rel="stylesheet" type="text/css" href="nazwapliku.css" />


Kolory



Text - fonts

Dołączamy czcionki np. z Google Fonts: Po wybraniu czcionki zaznaczam style, które mnie intesują, następnie jeśli będę umieszczać reguły css w pliku html wybieram link, jeśli będę umieszczać wszystko w osobnym pliku css wybieram @import, dodatkowo ukazuje się informacja jak opisać czcionkę.


Text - decoration

{text-decoration: none;} możesz też napisać tak a {text-decoration: underline line-through overline;} dla pod/prze/nadkreślenia jednocześnie letter-spacing: -2px/2px/20px odleglość liter od siebie word-spacing: 2px; text-indent: 20px odstęp akapitowy od strony text-align: center; tego nie musisz tłumaczyć :) text-decoration-color: #000; linie przekreślające mają kolor czarny text-shadow: 2pxw prawo 5pxdół 5pxrozmycie(4-10 najlepiej 5 przy 50 już nie widać) #000 kolor - zmienia kierunek, text-shadow: -2px w lewo -5px w górę


Actions

:hover, active, focus itd article:active {opacity .4;} transparentność zmiejszy się kiedy przytrzymam element klikając lifting effect article:hover {box/text-shadow: 5px 5px 20px rgba(0, 0, 0, .7);} płynne przejście transition: all 1s; w tym przypadku transition: box-shadow 1s, background #000 może być a nawet transition-delay: 2s;


Border radius

border radius rogi: 10px lewy górny, 20px prawy górny, 2px prawy dolny i 50px lewy dolny zgodnie z zegarem border-top-left-radius pierwsza wartosć określi zakrzywienie a druga zakrzywienie do w tym przypadku lewej strony dodatkowo wycięcie koła ze zdjęcie a w zasadzie zakrzywienie granic do kształtu koła .article-image {height:200px; width;200px: border-radius:100%;}
